## 1.명령어

#### status

### pull
```
git pull origin master
```
원격 저장소 변경 사항(이력)을 받아옵니다.
다른 작업 환경이나 위치에서 작업할 때, 혹은 공동 작업에서 타인이 commit해서 이력이 변경되었을 경우 등의 경우가 있습니다.
따라서 pull을 통해서 가져온 후, 작업을 진행하는 것이 좋습니다.
