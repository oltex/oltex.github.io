## 데이터 크기

1bit : 0 or 1  
8bit : 1byte  
1024byte : 1kb (2^10)  
1024kb : 1mb  
1024mb : 1gb
1024gb : 1tb

short : 2byte  
int : 4byte  
long : 4byte  
longlong : 8byte

## 데이터 표현 방식

10진수 8은 2진수로 0000 1000으로 표현된다.

숙제 : 숫자 10부터 0까지 2진수로 치환 10진수로 치환 공책에 써오기

### 정수 표현

unsigned :
- +나 -라는 부호(sign)가 없다. 양의 정수
- 자료형 앞에 unsigned 키워드를 붙여준다 ex)unsigned int

signed :
- +나 -라는 부호가 있다. 음의 정수, 양의 정수
- 자료형은 기본으로 signed로 표기된다.
